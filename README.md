# banco-json — Aroma Café

API mockada com JSON Server que alimenta o sistema **Aroma Café**.

## Rodar localmente
```bash
npm install
npm start
```
Servidor sobe em `http://localhost:3000`.

## Endpoints
- `/menu`
- `/pedidos`
- `/tipos_leite`
- `/tamanhos`
- `/opcionais`
- `/status_pedido`

## Deploy (Render)
- Build Command: `npm install`
- Start Command: `node index.js`
